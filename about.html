About This Guide

A free resource made by a homeschool mom, for homeschool families who love the outdoors.

🌿 What is this?

The Lake & Kenosha Homeschool Nature Guide is a free, community-built website for nature-loving homeschool families in Lake County, IL and Kenosha County, WI.

It gathers nature programs, outdoor events, and family-friendly places into one easy-to-search spot — because spending 45 minutes hunting through seven different park district websites to find one Saturday program is not how we want to spend our morning.

Everything here is open to all ages and all days of the week — perfect for flexible homeschool schedules.

🗓️ How does it stay up to date?

The Events section updates automatically every Monday morning. A script runs through the event calendars of Lake County Forest Preserves, local libraries, Kenosha County Parks, the Wisconsin and Illinois DNR, and more — then publishes the results here.

The Places section (trails, beaches, splash pads, etc.) is manually curated. These spots don't change often, but if something is out of date, please let me know!

📅 Update schedule: Events refresh every Monday at 6 AM. If you're checking mid-week, the list may be a few days old — but new events are always added each Monday.

📍 What's covered

Lake County IL sources

  • Lake County Forest Preserves
  • Illinois DNR events
  • Grayslake, Gurnee, Round Lake, Wauconda, Vernon Hills park districts
  • Lake County libraries (8 branches)
  • Visit Lake County aggregator

Kenosha County WI sources

  • Kenosha County Parks
  • Pringle Nature Center
  • Hawthorn Hollow
  • Kenosha Public Library
  • Wisconsin DNR events

Day trips to Chicago and Milwaukee are also included for museums, zoos, and botanical gardens worth the drive.

💚 Free & paid events

Both free and paid events are listed, always clearly labeled with a FREE or Paid badge. Use the Free only filter if you want to see just no-cost events.

Many of the best programs in this area — especially through the Lake County Forest Preserves, public libraries, and Kenosha County Parks — are completely free.

✉️ Suggest a place or event

Know a great trail, playground, splash pad, or recurring program that should be on here? Found something out of date or incorrect?

This site lives on GitHub — you can open an issue or pull request if you're comfortable with that, or just reach out directly. Community contributions make this better for everyone.

The source code and all data files are public, so anyone can see exactly how it works.

🛠️ Technical notes

This is a simple static website hosted for free on GitHub Pages. No ads, no tracking, no accounts, no paywalls.

  • Events data: automatically scraped and committed every Monday via GitHub Actions
  • Places data: hand-curated JSON file, updated as needed
  • Map: Leaflet.js with OpenStreetMap tiles (both free and open source)
  • No cookies, no analytics, no external fonts
